如何使用这张CD
==================

请将CD放入CD-ROM驱动器，启动界面将自动出现在屏幕上。如果改界面没有出现，
请打开根目录下的StartCD.exe文件



本CD的内容
==================

 -  原书中的示例代码
 -  示例代码的安装程序
 -  本书的补充内容（英文）


原书中的示例代码
---------------------------

可以浏览本CD的Sample目录直接查看示例代码。这些代码都是按章组织的。记住，
如果手动地将这些文件复制到硬盘上，那么它们的属性就是只读的。要清楚只读
属性，请右击文件图标，从出现的菜单中选择【属性】，取消【只读】前的复选
框即可。


示例的安装程序
---------------------------------

本书示例文件的安装程序位于:

    Setup\Setup.exe

This setup program copies the files to your hard disk and 
clears the read-only flag on the files. You must have
administrative privileges to run this setup program on 
Microsoft Windows NT 4.0, Microsoft Windows 2000, or Microsoft
Windows XP. To uninstall the files make the appropriate selection
from Add/Remove Programs in the Control Panel.
安装程序将示例文件复制到硬盘上并清楚文件的只读属性。您必须具备
管理员身份才能运行安装程序，安装程序适合的操作系统是Microsoft 
Windows NT 4.0, Microsoft Windows 2000, 或MicrosoftWindows XP。
如果要卸载这些文件，请在控制面板上的【添加/删除程序】中进行选择。
